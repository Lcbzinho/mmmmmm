# mmmmmm

Aplicação Flask simples com templates HTML.

## Estrutura

- app.py
- templates/
	- base.html
	- index.html
- static/
	- styles.css
- requirements.txt

## Como executar

1. Crie e ative um ambiente virtual (opcional).
2. Instale as dependências:

	 pip install -r requirements.txt

3. Rode a aplicação:

	 python app.py

4. Acesse: http://127.0.0.1:5000/
